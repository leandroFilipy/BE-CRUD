# 🚀 BE-CRUD

Bem-vindo ao **BE-CRUD**! Este é um projeto backend CRUD (Create, Read, Update, Delete) desenvolvido para facilitar o gerenciamento de dados de forma simples e eficiente. 🔥

## 💡 Sobre o Projeto

Este repositório foi criado com o objetivo de demonstrar as operações básicas de CRUD em uma API backend. Ele pode ser utilizado como base para projetos de aprendizado, testes ou como ponto de partida para sistemas mais complexos.
